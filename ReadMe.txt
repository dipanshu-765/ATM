RSA_General file has encryption and decryption functions for RSA. 
This file is used to import these functions in ATM file.
It also has key generation function for RSA.

Generate_Keys generates and displays public and private key value pairs for RSA.
We're using one such pair. The key values may be changed in ATM.py file to key value pair generated
through Generate_Keys file.

Admin file is the file for admin to create new accounts or update accounts in the database.

ATM file is the file which performs the Withdrawal, Check Balance and Change PIN functions.
